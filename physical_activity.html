How do I join the Step Challenge?
  1. From the HealthSlate home screen:
    1. iOS device: select the Team icon near the bottom of the screen.
    2. Android device: Select the My Coaches icon near the top right corner of the screen.
  2. Scroll down and select Join Now.
What apps are compatible with the HealthSlate app?
  • The HealthSlate app is compatible with the Google Fit app, Apple HealthKit app, Fitbit app and Nokia Health Mate app.
What step tracking devices are compatible with the HealthSlate app?
  • A step tracking device must be able to transmit steps to either the Google Fit app, Apple HealthKit app or Fitbit app for the steps to display from the HealthSlate app.
How are my steps calculated in the Step Challenge?
  • The Step Challenge displays the average number of steps taken since Monday.
How do I input a custom activity?
  1. From the HealthSlate app home screen:

    1. iOS device: select the icon.

    2. Android device: select the icon.
  2. Select Activity

  3. Select the small grey pencil icon directly under Type of activity.

  4. Select the type of activity you'd like to log or select Other.
  5. Input the precise activity you are participating in. You can also input the number of minutes, intensity level, feeling, and any notes.
How do I add a new goal?
  1. From the HealthSlate home screen:
    1. iOS device: select More
    2. Android device: select the menu icon in the top left corner of the screen.

  2. Select Goals.
  3. Select the +Goal icon in the top right corner of the screen.
  4. 4. Select the type of goal you would like to input.
  5. Select a goal from the available list or create your own goal by selecting Create my own Goal.
  6. If you choose Create my own Goal input a name for the goal.
  7. Select the days of the week, followed by the time of the day that you’d like the reminder to occur on. You can also turn the reminder on or off using the Reminder On/Off slider.
  8. Select the Save button to save the goal.
