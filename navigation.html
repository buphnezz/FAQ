Navigation / Interaction

Setting Calorie Limit

  1. From the HealthSlate home screen:
    • iOS Device: select Moreand select Settings
    • Android Device: select the menu and select Settings
  2. Select Calorie Budget and Logging.
  3. Select Daily Calorie Budgetand input your new Daily Calorie Budgetand select Save.


Viewing weekly weight chart

  1. From the HealthSlate home screen:
    • iOS device: select More
    • Select Progress and Logs
    • Android device: select the menu icon in the top left corner of the screen.
    • Select Progress and Logs.
  2. Select Progress and Logs.
  3. Select Weight.
  4. You will now be able to view your weekly weight chart(s).
Messaging your coach from a desktop or laptop

  1. From the HealthSlate home screen select the My Group icon in the top right corner of the screen.

  2. To send a message directly to your coach type the @ symbol followed by your coach's name and what you'd like to message to your coach.

  3. If you'd like to send a message to all coaches, simply type your message without using the '@' symbol.

Favorite a meal
  1. To favorite a meal, from the HealthSlate home screen:
    • iOS device: select the Food icon on the bottom of the screen.
    • Android device: select the Food icon from the top of the screen.
  2. Navigate to the meal you would like to favorite and select the meal.
  3. Select the star icon directly below the image of the meal.
Post a message to the Team Wall
  • iOS device: select the Wallicon near the bottom of the screen.
  • Android device: Select the My Group icon near the top right corner of the screen.
    1. Type the message you would like to post into the Share what's on your mindfield.
Contact Tech Support
  1. From the HealthSlate Home page, select the Team icon.
  2. Type @Support, followed by what you would like to message to the tech support into the Type a message...chat box.
Contact your coach
  1. From the HealthSlate Home Page, select the Team icon.
  2. Select My Coaches.
  3. Type @ followed you your coach's name and input what you would like to message to your coach into the Type a message...chat box.
Setting a Weight Goal and/or Target Weight
  1. From the HealthSlate home screen:
    • iOS device: select Moreicon and select Settings
    • Android device: select the menu icon in the top left corner of the screen and select Settings.
  2. Select Weight Goal and Logging.
  3. Select Target Weight.
  4. Imput your desired Target Weight, select Save.(Your Target Weightacts as your weight goal.)
