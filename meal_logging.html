Meal Logging

Input a meal using Free Form Text
  1. From the HealthSlate app home screen:
    1. iOS device: select the button.

    2. Android device: select the button.
  2. Select Meals.
  3. Select the meal you would like to log.
  4. Select Free Form Text.
  5. Select the microphone button to speak the items you consumed for your meal and select Submit.
  6. he items and their respective calories should be displayed, scroll to the bottom of the screen and select Save.
Favorite a meal
  1. To favorite a meal, from the HealthSlate home screen:
    • iOS device: select the Food icon on the bottom of the screen.
    • Android device: select the Food icon from the top of the screen.
  2. Navigate to the meal you would like to favorite and select the meal.
  3. Select the star icon directly below the image of the meal.

Setting your calorie budget

  1. From the HealthSlate home screen:
    • iOS Device: select Moreand select Settings
    • Android Device: select the menu and select Settings
  2. Select Calorie Budget and Logging.
  3. Select Daily Calorie Budgetand input your new Daily Calorie Budgetand select Save.
