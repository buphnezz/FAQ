Meal Logging

Input a meal using Free Form Text
  1. From the HealthSlate app home screen:
    1. iOS device: select the button.

    2. Android device: select the button.
  2. Select Meals.
  3. Select the meal you would like to log.
  4. Select Free Form Text.
  5. Select the microphone button to speak the items you consumed for your meal and select Submit.
  6. he items and their respective calories should be displayed, scroll to the bottom of the screen and select Save.
Favorite a meal
  1. To favorite a meal, from the HealthSlate home screen:
    1. iOS device: select the Food icon on the bottom of the screen.
    2. Android device: select the Food icon from the top of the screen.
  2. Navigate to the meal you would like to favorite and select the meal.
  3. Select the star icon directly below the image of the meal.
Delete a Meal
  1. From the HealthSlate home screen, scroll down and select Log Book.
    1. iOS device: select the Meals tab near the top of the screen.

    2. Android device: select Meal from the drop down menu near the top right corner of the screen.
  2. Navigate to, and select the meal you'd like to delete.
  3. Select Delete Meal.
Setting Calorie Budget

  1. From the HealthSlate home screen:
    1. iOS Device: select More .

    2. Android Device: select the menu
  2. select Settings
  3. Select Calorie Budget and Logging.
  4. Select Daily Calorie Budgetand input your new Daily Calorie Budgetand select Save.
