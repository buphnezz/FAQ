Account and Profile

How do I join the Step Challenge?
  1. From the HealthSlate app home screen:
    1. iOS device: select the Team icon near the bottom of the screen.
    2. Android device: Select the My Coaches icon near the top right corner of the screen.
  2. Scroll down and select Join Now.
How do I upload or change my profile picture?
  1. From the HealthSlate app home screen:
    1. iOS device: select More
    2. Android device: select the menu icon in the top left corner of the screen.
  2. Select Settings.
  3. Select Profile.
  4. Select Profile Picture.
  5. Select the small grey pencil icon in the top right corner.
  6. Select Take Photo or Choose Existing Photo
  7. Once you have selected the photo that you want, select Save.
How do I cancel my HealthSlate account?
  • Please message tech support with this request.
How do I change My Motivational picture and phrase?
  1. From the HealthSlate app home screen:
    1. iOS device: select More
    2. Android device: select the menu icon in the top left corner of the screen.
  2. Select Settings.
  3. Select Profile.
  4. Select My Motivation.
  5. To change the My Motivation picture, Select the small grey pencil icon in the top right corner.
  6. Select Take Photo or Choose Existing Photo
  7. To Change the My Motivational Phrase, type the phrase into the text area.
  8. Once you have the motivational photo and motivational phrase that you want, select Save.
