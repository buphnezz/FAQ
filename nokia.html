Nokia Scale Questions

Nokia Scale Setup

  1. Pull the tab out of the battery compartment if it’s the first time you install your Nokia Body Scale.
  2. Download and install the Nokia Health Mate® app. Then Open the Health Mate® app and select Get Started.
  3. Select SET UP MY DEVICE
  4. Select Scales, then select Body.
  5. Select Install
    1. You may need allow the Nokia Health Mate app to access your position. Select Allow.
    2. Health Mate may need to turn on your Bluetooth. Select Allow.
  6. On the Nokia Scale, press the button at the back of the scale for three seconds. Once the scale reads Start app, select Next on the Health Mate® app.
  7. Once your Nokia Scale is detected, select Next.
  8. Enter your first name, last name, Gender, and Birthday. Select Next.
  9. Enter your height and weight and select Next.
  10. If you would like to install your Nokia Scale on Wi-Fi perform the following steps:
    If you would like to install your scale on Bluetooth, click here IF YOU CAN READ THIS FIX THE LINK!
    1. Select Set up Wi-Fi connection.
    2. Select the Wi-Fi network you want to use or select Manual configuration.
    3. Enter the password of the Wi-Fi network.
    4. Select Next.
    Note: You will need to update the Wi-Fi configuration if you change the network name or password.
  11. Select Done.
  12. Select Done.
    13. For instruction on how to sync the weights to the HealthSlate app click here IF YOU CAN READ THIS FIX THE LINK

Enable Nokia Scale to Transmit Weight to the HealthSlate App

  1. Open the HealthSlate app
  2. From the HealthSlate home screen:
    1. IOS device: select More
    2. Android device: select the menu icon in the top left corner of the screen.
  3. Select Settings
  4. Select Manage Device Sync.
  5. Select Manage Nokia Scale.
  6. Check the box next to Nokia Scale and select Authorize.
  7. Enter the email and password you used with your Health Mate® account and select Sign in.
  8. Select Allow.
  9. You will be redirected back to the Settings screen. Your weights should now be synced between your Nokia Scale and your HealthSlate app.
Change Nokia Scale from Kilograms (kg) to Pounds (lbs)

  1. Press and hold the button at the back of your scale for a one second.
  2. Press the button again to change the unit.
  3. The unit change is confirmed a few seconds after you stop pressing the button.
