Messaging and Communication

Message your coach from the HealthSlate app

  1. From the HealthSlate home page, select the Team icon.
  2. Select My Coaches.
  3. Type @, followed you your coach's name and input what you would like to message to your coach into the Type a message... chat box.

Message tech support from the HealthSlate app

  1. From the HealthSlate home page, select the Team icon.
  2. Select My Coaches.
  3. Type @Support, followed by the message you would like to send into the Type a message...chat box.
Post a message to the team wall from the HealthSlate app

  1. From the HealthSlate home screen:
    1. iOS device: select the Wall icon near the bottom of the screen.
    2. Android device: Select the My Group icon near the top right corner of the screen.
  2. Type the message you would like to post into the Share what's on your mind field.
How to change or turn off push notifications and reminders

  1. From the HealthSlate app home screen:

    1. iOS device: select More
    2. Android device: select the menu icon in the top left corner of the screen.

  2. Select Settings.
  3. Select Calorie Budget and Logging.
  4. Select the bell icon to turn on or off reminders to log meals.
  5. Select the meal you would like to change the time of.
  6. Select Weight Goal and Logging.
    1. iOS device: select the Weigh myself toggle to turn on and off the daily weigh in reminder notification.
      Select the Weight in reminder to change time the weigh in reminder notification is sent.
    2. Android device: select the bell icon to turn on and off the daily weigh in reminder.
      Press the time to change time the weight in reminder notification is sent.
