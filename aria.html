Weight Tracking: Fitbit Aria Scale

How do I set up the FitBit Aria scale?

Please visit fitbit.com/setup/aria and follow the instructions to setup your Aria scale.

How do I sync my FitBit Aria scale with the HealthSlate app?
  1. From the HealthSlate app home screen:
    1. iOS Device: select More

    2. Android Device: select the menu icon in the top left corner of the screen.
  2. Select Settings.
  3. Select Manage Device Sync.
  4. Select Manage Fitbit devices.
  5. Ensure there is a check mark in the box next to Aria, and select Authorize.
  6. Login to your Fitbit account to confirm the authorization process
  7. Once logged in ensure there are check marks in the boxes next to profile and weight, and select Allow.
