Aria scale


Requirements to use scale

  • BodyTrace Scale
    • There are no requirements to use the BodyTrace scale. Simply stand on the BodyTrace scale and your weight will transmit to the HealthSlate app.

  • Nokia (Withings) Body scale
    • The Nokia Body scale must be linked to the Nokia Health Mate app before weight will transmit to the HealthSlate app

  • Fitbit Aria Scale
    • The Aria scale must be linked to a Fitbit account before weight will transmit to the HealthSlate app.
Setting a weight goal and/or target weight

  1. From the HealthSlate home screen
    1. iOS device: select More
    2. Android Device: select the menu icon in the top left corner of the screen.
  2. Select Settings.
  3. Select Weight Goal and Logging.
  4. Select Target Weight..
  5. Input your desired Target Weight, select Save. (Your Target Weight acts as your weight goal).
View weekly weight chart

  1. From the HealthSlate home screen
    1. iOS device: select More

    2. Select Progress and logs.

    3. Android Device: select the menu icon in the top left corner of the screen.
    4. Select Progress and logs.

  2. Select Weight.
  3. You will now be able to view your weekly weight chart(s).
Estimated time to receive scale

The scale should arrive around 10 business days after joining the HealthSlate program.
